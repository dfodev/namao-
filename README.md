# NaMão - Conectando Profissionais do Seu Bairro

Plataforma web para conectar prestadores de serviços locais a clientes via WhatsApp.

## 🚀 Funcionalidades

- Cadastro de clientes e prestadores de serviços
- Busca por bairro e categoria
- Sistema de avaliações e favoritos
- Dashboard com métricas para prestadores
- Portfólio com imagens
- Contato direto via WhatsApp
- Painel administrativo

## 🛠️ Tecnologias

- Python 3.x
- Flask
- SQLAlchemy
- SQLite
- Bootstrap 5
- Font Awesome
- Jinja2

## 📋 Pré-requisitos

- Python 3.8 ou superior
- pip (gerenciador de pacotes Python)
